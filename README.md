# Magnet Hunter - 磁力链接一键提取器

**这是一个专为 Microsoft Edge 浏览器打造的高效插件，能够一键提取所有打开网页内的磁力链接（可事先自动唤醒所有已打开的标签页，并运用并行处理技术一次性提取所有页面中的磁力链接）。所有数据处理均在本地进行，不会上传任何信息。**

它解决了由于标签页长时间未被点击而休眠导致脚本无法提取磁力链的问题，支持多页、单页多链接提取，最终将结果汇总导出为 TXT 文档，极其适合批量收集资源的场景。

## 📂 文件结构

项目文件应包含以下文件：
- `manifest.json` (配置文件)
- `background.js` (核心后台逻辑)
- `popup.html` (插件弹窗界面)
- `popup.js` (弹窗交互逻辑)

## 🛠️ 安装方法

由于本插件未发布到应用商店，请按照以下步骤以“开发者模式”安装：

1. **下载代码**：点击 GitHub 页面右上角的 `Code` -> `Download ZIP`，下载并解压到本地文件夹（例如 `D:\MagnetHunter`）。
2. **打开扩展中心**：在 Edge 浏览器地址栏输入 `edge://extensions/` 并回车。
3. **开启开发者模式**：打开页面左侧（或顶部）的 **“开发人员模式” (Developer mode)** 开关。
4. **加载插件**：点击出现的 **“加载解压缩的扩展” (Load unpacked)** 按钮，选择第 1 步中解压的文件夹。

## 🚀 使用方法

本插件采用 **“极速激活 - 并行提取”** 的双阶段工作流：

1. **准备页面**：在浏览器中打开你需要提取磁力链接的所有网页（打开即可，无需等待完全加载，允许页面处于后台）。
2. **启动插件**：点击浏览器右上角的拼图图标，选择 **Magnet Hunter**，点击 **“启动两阶段提取”** 按钮。
3. **阶段一（自动激活）**：
   - 此时请**松开鼠标**。插件会自动快速切换每一个标签页。
   - *目的：将休眠的标签页唤醒至内存中，以便脚本读取。*
4. **阶段二（确认并提取）**：
   - 遍历结束后，浏览器会切回起始页，并弹出一个确认对话框。
   - 点击 **“确定”**，插件将瞬间并行扫描所有页面。
5. **保存结果**：浏览器弹出“另存为”窗口，选择保存位置（如桌面），即可获得包含所有链接的 `.txt` 文件。如果没有磁力链被发现则不弹出“另存为”窗口。

## ⚠️ 注意事项

- **关于权限**：插件需要访问 `tabs` 和 `<all_urls>` 权限以读取网页内容，所有数据处理均在本地进行，不会上传任何信息。
- **关于保存位置**：受限于现代浏览器的安全机制，插件不能静默写入本地磁盘，因此每次提取完成都会弹出系统的“另存为”窗口供您确认。
